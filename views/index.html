Image Search Abstraction Layer

How to Use

Node search engine based on google that responds with a json object containing a url and other information on each result. To search use this homepage with /search/yourSearch where yourSearch is whatever images your looking for. Can request different result pages by adding ?page=1 which can be set to any positive integer. This project uses the Google Custom Search Engine through a node library called google-images. Can get search history at /history and recieve the history of searches in json formatting. Simple history database with mongoDB courtesy of mlab.

A few example calls:

  • https://image-abs.glitch.me/search/dalmatian puppy
  • https://image-abs.glitch.me/search/kitten?page=1

The json response structure: 

     {
      "type": "image/jpeg", "width": 480, "height": 360, "size": 19463, "url": "https://examplelink/image.jpg",
      "thumbnail": {"url": "https://examplelink", "width":129, "height":97},
      "description": "Example Description", "parentPage": "https://parentlink.example.com"
     }

Can also view search history at /history;

https://image-abs.glitch.me/history
{"search": "dalmation puppy", "page": 1, "time": 1508264514814},
{"search": "kitten", "page": 1, "time": 1508265610124}

Time is in milliseconds from Javascript Date object
